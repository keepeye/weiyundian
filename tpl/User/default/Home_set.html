当用户发送 "首页" ,系统会自动回复一条包含"回复标题"和"内容介绍"、"回复图片的"图文消息，点击后会打开首页网页，首页上显示的是分类导航，你需要通过分类管理添加不同的导航链接才有显示，记得先在模板管理中选择一个喜欢的首页模板。

*关键词： 首页 或者 home —— 当用户输入该关键词时，将会触发此回复。

宽640高960
